### Added
- Y key (0.4s hold): Anchor Grid popup
- Grid size: 3x3 to 7x7 configurable
- Custom anchor points (3 slots)
- Copy/Paste anchor
- Composition mode / Selection mode toggle
- Mask recognition mode
- CEP Settings panel
- Double-tap Y: Toggle mode (click to apply)

### Install Paths
- Plugin: `MediaCore/AnchorSnap.aex`
