## 그리드 영역 (Grid Area)

| 용어 | 영어 | 설명 |
|------|------|------|
| **셀** | Cell | 그리드의 각 칸. 클릭하면 해당 위치로 앵커 설정 |
| **마크** | Mark | 셀 안에 그려지는 앵커 위치 표시 (십자가/L자/T자) |
| **마크 원 (도트)** | Dot | 마크 중앙의 작은 원 |
| **구분선** | Grid Line | 셀 사이의 회색 경계선 |
| **호버 글로우** | Hover Glow | 마우스 올렸을 때 나타나는 발광 효과 |
| **셀 배경** | Cell Background | 셀의 배경색 (투명도 조절 가능) |

## 마크 타입 (Mark Types)

| 타입 | 위치 | 모양 | 변수명 |
|------|------|------|--------|
| **코너 마크** | 4개 모서리 (좌상, 우상, 좌하, 우하) | L자 형태 └ ┘ ┐ ┌ | `cornerLen` |
| **엣지 마크** | 가장자리 중간 (상/하/좌/우) | T자 형태 ┴ ┬ ┤ ├ | `edgeLen` |
| **센터 마크** | 내부 셀 (가운데 영역) | + 십자 형태 | `centerLen` |

### 마크 관련 변수
```cpp
int edgeOffset;   // 코너/엣지 마크가 중앙에서 떨어진 거리
int cornerLen;    // 코너 마크의 선 길이
int edgeLen;      // 엣지 마크의 선 길이
int centerLen;    // 센터 마크의 선 길이
int radius;       // 마크 원(도트)의 반지름
int hoverRadius;  // 호버 글로우의 반지름
```

## 아이콘 (Icons)

### 왼쪽 패널 (Left Panel)

| 아이콘 | 이름 | 설명 | 변수명 |
|--------|------|------|--------|
| ① | 커스텀 앵커 1 | 저장된 앵커 프리셋 1 | `OPT_CUSTOM_1` |
| ② | 커스텀 앵커 2 | 저장된 앵커 프리셋 2 | `OPT_CUSTOM_2` |
| ③ | 커스텀 앵커 3 | 저장된 앵커 프리셋 3 | `OPT_CUSTOM_3` |
| ④ | Copy | 현재 앵커 비율 복사 | `OPT_COPY_ANCHOR` |
| ⑤ | Paste | 복사된 앵커 비율 붙여넣기 | `OPT_PASTE_ANCHOR` |

### 오른쪽 패널 (Right Panel)

| 아이콘 | 이름 | 설명 | 변수명 |
|--------|------|------|--------|
| ⑥ | 컴포지션/셀렉션 | 셀렉션(파랑)/컴포지션(주황) 모드 토글 | `OPT_COMP_MODE` |
| ⑦ | 마스크 ON/OFF | 마스크 경계 인식 ON/OFF | `OPT_MASK_MODE` |
| ⑧ | 설정 (톱니바퀴) | CEP 설정 패널 열기 | `OPT_SETTINGS` |

## 색상 (Colors)

| 용어 | 영어 | 용도 | 색상 값 |
|------|------|------|---------|
| **파란색** | Blue | 셀렉션 모드, 마스크 ON, 활성 상태 | `#4A9EFF` |
| **주황색** | Orange | 컴포지션 모드 | `#FFB44A` |
| **회색** | Dark Gray | 비활성 상태, 구분선 | `#464646` |
| **글로우 색상** | Glow | 호버 시 발광 | Inner → Mid → Outer |

## CEP 설정 (Settings)

| 설정 | 영어 | 설명 |
|------|------|------|
| **그리드 크기** | Grid Size | 행/열 개수 (3~7) |
| **그리드 스케일** | Grid Scale | 전체 크기 배율 (-40% ~ +40%) |
| **마크 투명도** | Mark Opacity | 마크 표시의 투명도 |
| **셀 투명도** | Cell Opacity | 셀 배경의 투명도 |

---

## 모드 (Modes)

| 모드 | 영어 | 설명 |
|------|------|------|
| **셀렉션 모드** | Selection Mode | 선택된 레이어 기준으로 앵커 계산 |
| **컴포지션 모드** | Composition Mode | 컴포지션 전체 기준으로 앵커 계산 |
| **마스크 인식** | Mask Recognition | 마스크 경계를 기준으로 앵커 계산 |
| **홀드 모드** | Hold Mode | Y 키를 누르고 있는 동안 그리드 표시 |
| **토글 모드** | Toggle Mode | Y~Y (빠르게 두 번)로 클릭까지 그리드 유지 |
