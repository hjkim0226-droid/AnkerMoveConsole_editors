1/*****************************************************************************
2 * TextUI.h
3 *
4 * Native Windows UI for Anchor Snap - Text Options Module
5 * Provides quick text layer property adjustment
6 * Trigger: D → T sequence (D key + T key within 500ms)
7 *
8 * Features:
9 * - Drag scrubbing for numeric values (Size, Tracking, Leading, StrokeWidth)
10 * - Double-click for direct input (inline edit with style toggle)
11 * - Color picker for Fill/Stroke
12 * - Font family/style dropdown
13 * - Paragraph alignment buttons
14 *
15 * Key mechanism: WS_EX_NOACTIVATE for drag mode (preserves AE text input),
16 * temporarily removed for edit mode (allows keyboard input)
17 *****************************************************************************/
